  <action>
Configure Drizzle ORM with Neon Postgres serverless driver and create database schema with users and credentials tables per CONTEXT.md decisions D-02 and D-03.

**Concrete implementation:**

1. Install dependencies:
   ```bash
   npm install drizzle-orm@latest @neondatabase/serverless@latest
   npm install -D drizzle-kit@latest
   ```

2. Create .env.local with:
   ```
   DATABASE_URL=postgresql://[user]:[password]@[host]/[database]?sslmode=require
   AUTH_SECRET=generated-secret-min-32-chars
   SHAGGYOWL_BASE_URL=https://app.shaggyowl.com
   ```
   Use `openssl rand -base64 32` to generate AUTH_SECRET.

3. Create .env.example (NO secrets, just keys):
   ```
   DATABASE_URL=
   AUTH_SECRET=
   SHAGGYOWL_BASE_URL=https://app.shaggyowl.com
   ```

4. Create drizzle.config.ts:
   ```typescript
   import type { Config } from 'drizzle-kit';

   export default {
     schema: './lib/db/schema/index.ts',
     out: './drizzle',
     dialect: 'postgresql',
     dbCredentials: {
       url: process.env.DATABASE_URL!,
     },
   } satisfies Config;
   ```

5. Create lib/db/schema/users.ts with Auth.js v5 compatible schema:
   ```typescript
   import { pgTable, text, timestamp, uuid } from 'drizzle-orm/pg-core';

   export const users = pgTable('users', {
     id: uuid('id').defaultRandom().primaryKey(),
     name: text('name'),
     email: text('email').notNull().unique(),
     emailVerified: timestamp('email_verified', { mode: 'date' }),
     image: text('image'),
     password: text('password').notNull(), // Hashed with bcrypt
     createdAt: timestamp('created_at').defaultNow().notNull(),
     updatedAt: timestamp('updated_at').defaultNow().notNull(),
   });
   ```

6. Create lib/db/schema/credentials.ts (D-02, D-03):
   ```typescript
   import { pgTable, text, timestamp, uuid, boolean } from 'drizzle-orm/pg-core';
   import { users } from './users';

   export const credentials = pgTable('credentials', {
     id: uuid('id').defaultRandom().primaryKey(),
     userId: uuid('user_id').notNull().references(() => users.id, { onDelete: 'cascade' }),
     pilactiveEmail: text('pilactive_email').notNull(),
     pilactivePassword: text('pilactive_password').notNull(), // TODO: encrypt in Phase 4
     sessionToken: text('session_token'), // codice_sessione from ShaggyOwl
     tokenExpiresAt: timestamp('token_expires_at', { mode: 'date' }),
     isActive: boolean('is_active').default(true),
     createdAt: timestamp('created_at').defaultNow().notNull(),
     updatedAt: timestamp('updated_at').defaultNow().notNull(),
   });
   ```

7. Create lib/db/schema/index.ts:
   ```typescript
   export * from './users';
   export * from './credentials';
   ```

8. Create lib/db/index.ts:
   ```typescript
   import { drizzle } from 'drizzle-orm/neon-http';
   import { neon } from '@neondatabase/serverless';
   import * as schema from './schema';

   if (!process.env.DATABASE_URL) {
     throw new Error('DATABASE_URL environment variable is not set');
   }

   const sql = neon(process.env.DATABASE_URL);
   export const db = drizzle(sql, { schema });
   ```

9. Add package.json scripts:
   ```json
   {
     "scripts": {
       "db:generate": "drizzle-kit generate",
       "db:push": "drizzle-kit push",
       "db:studio": "drizzle-kit studio"
     }
   }
   ```

10. Push schema to Neon database:
    ```bash
    npm run db:push
    ```

**DO NOT:**
- Store passwords in plain text (use bcrypt for user.password, defer encryption for pilactivePassword to Phase 4)
- Use Prisma (violates D-01)
- Skip sessionToken field in credentials table (violates D-03)
- Use separate databases for users and credentials (violates D-02 normalized design)

**User setup required:**
Before running db:push, user must:
1. Create Neon Postgres database at https://neon.tech
2. Copy connection string to .env.local as DATABASE_URL
3. Ensure connection string includes ?sslmode=require
